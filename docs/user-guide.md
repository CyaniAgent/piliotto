# 安装与使用

本指南帮助你快速上手 PiliOtto。

## 基本使用

### 首页

打开应用后进入首页，包含以下 Tab：

- **推荐** — 个性化推荐内容
- **热门** — 热门视频榜单
- **排行** — 各分区排行榜

### 视频播放

点击视频卡片进入播放页面：

- **播放控制** — 点击屏幕显示控制栏
- **弹幕开关** — 右上角弹幕按钮
- **倍速播放** — 右下角倍速选择
- **全屏播放** — 右下角全屏按钮
- **画中画** — 支持后台播放

### 搜索

点击顶部搜索框：

- 输入关键词搜索视频
- 查看搜索历史
- 支持筛选排序

### 个人中心

点击底部「我的」Tab：

- 查看个人信息
- 管理收藏夹
- 查看历史记录
- 应用设置

## 设置

### 主题设置

- **深色/浅色模式** — 跟随系统或手动切换
- **主题色** — 自定义强调色
- **字体大小** — 调整界面字体

### 播放设置

- **默认倍速** — 设置默认播放速度
- **长按倍速** — 长按加速播放
- **手势控制** — 音量/亮度/进度手势

### 其他设置

- **导航栏定制** — 自定义底部导航
- **首页 Tab 定制** — 自定义首页标签

## 常见问题

### 视频无法播放？

1. 检查网络连接
2. 尝试切换视频源
3. 清除应用缓存

### 登录失败？

1. 检查账号密码是否正确
2. 尝试重新登录
3. 检查网络连接

### 应用闪退？

1. 清除应用缓存
2. 更新到最新版本
3. 反馈问题到 GitHub Issues
